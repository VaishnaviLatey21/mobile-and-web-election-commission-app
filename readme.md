# Installation & Setup

### Install nodejs or update to the latest version if already installed

### Backend Setup

Navigate to the backend folder:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Start the server:

```bash
npx nodemon
```


### Frontend Setup

Navigate to the frontend folder (in a new terminal):

```bash
cd frontend
```

Install dependencies:

```bash
npm install --force
or
npm install --legacy-peer-deps
```

Start the React app:

```bash
npm run dev
```
